<b>Face++</b>

https://www.faceplusplus.com.cn/license-plate-recognition/ 
检测图片中的车牌，并返回车牌边框坐标、车牌号码、车牌颜色等信息。支持识别各种位置、白天以及夜间的车牌识别。
- 精准识别
- 适应不同光照场景
- 适应倾斜/模糊等场景
